🌟 O projektu

Tento projekt není jen hudební přehrávač – je to Komunikační stanice 5. generace pro tvou hudební flotilu. LCARS-inspirovaný audio přehrávač kombinuje robustní JavaScript moduly s moderními webovými technologiemi.

🗣️ Hlasové ovládání (CZ)
🎨 LCARS design
📱 Media Session API
⚡ 60+ FPS

✨ Klíčové vlastnosti

🎤 Interakce a ovládání

  • Hlasové příkazy v češtině: „Počítači, další / pauza / hlasitost 50…“
  • Media Session API: ovládání z notifikací, displeje v autě, BT ovladače
  • Mini Přehrávač: plovoucí okno, drag & drop, kompaktní UI

🎼 Správa playlistu

  • Drag & Drop pořadí, přejmenování, oblíbené
  • Časové záložky, rychlé vyhledávání
  • Témata: Dark / Light / Neon / Custom

🛠️ Technické subsystémy

  • Auto-Fade (crossfade) pro plynulé přechody
  • Detekce odpojení (Bluetooth / 3.5mm jack) → auto pauza
  • Firebase synchronizace: nastavení UI, záložky, oblíbené
  • Performance monitor: FPS & paměť (UI i konzole)

🚀 Rychlý start

„Počítači, Engage!“

# 1. Klonuj repozitář
git clone https://github.com/jirka22med/star-trek-hudebni-prehravac-vylepsen.git

# 2. Vstup do složky
cd star-trek-hudebni-prehravac-vylepsen

# 3. Uprav playlist
nano myPlaylist.js

# 4. Otevři v prohlížeči
open index.html

Nastavení playlistu

// myPlaylist.js
window.tracks = [
  { src: 'https://www.dropbox.com/…/song.mp3?dl=1', title: 'Nová-skladba' }
];

Tip: u Dropbox odkazů vždy končete ?dl=1 pro přímé stažení/stream.

🛡️ Troubleshooting

„Skladba se nenačte“

  1. URL končí na ?dl=1
  2. Link je veřejný
  3. Soubor je .mp3/.ogg/.wav
  4. Bez překlepů

CORS chyby

Dropbox s ?dl=1 vrací správné CORS hlavičky.

iOS Safari limity

  • Preferuj MP3, bitrate 128–192 kbps
  • Některé akce vyžadují uživatelské gesto
